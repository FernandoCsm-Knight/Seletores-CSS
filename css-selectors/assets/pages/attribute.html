Seletores de atributo em CSS

[attribute]attribute selector

Seleciona todos os elementos com o atributo. Exemplo:

[aria-details] {}

[attribute=value][attribute=value] selector

Seleciona todos os elementos com o atributo e valor específicos. Exemplo:

[aria-details="exemplo-a"] {}

[attribute~=value][attribute~=value] selector

Seleciona todos os elementos que contém o exato valor. Exemplo:

[aria-details~="atributo"] {}

[attribute|=value][attribute|=value] selector

Seleciona todos os elementos que começam com o exato valor especificado. Exemplo:

[aria-details|="atributo"] {}

[attribute^=value][attribute^=value] selector

Seleciona todos os elementos que começam com o valor especificado. Exemplo:

[aria-details^="x"] {}

[attribute$=value][attribute$=value] selector

Seleciona todos os elementos que terminam com o valor especificado. Exemplo:

[aria-details$="y"] {}

[attribute*=value][attribute*=value] selector

Seleciona todos os elementos que contém o valor. Exemplo:

[aria-details*="z"] {}
