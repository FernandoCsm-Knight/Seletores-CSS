Pseudo Classes

: Pseudo Classes

É utilizada para definir um estado especial de um elemento. Exemplo:

.pseudo:pseudo-classe {}

:hover :hover

É utilizada para selecionar o elemento ao passar o mouse. Exemplo:

.pseudo-hov:hover {}

:nth-child() :hover

É utilizada para selecionar o enésimo elemento da sequencia. Exemplo:

.box-child:nth-child(2) {}
.box-child:nth-child(2n) {}
.box-child:nth-child(odd) {}

Links Estilizando links

Estilizando a tag "a" ao passar o mouse, clicar e histórico. Exemplo:

.a:link {}
.a:hover {}
.a:visited {}
.a:active {}

:: Pseudo Elementos

É utilizada para estilizar uma parte especifica do elemento. Exemplo:

.pseudo-e::after {}

::after ::after

.after > p::after {}
