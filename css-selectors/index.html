Seletores em CSS

* Universal selector

Seleciona todos os seletores CSS. Exemplo:

* {}

# id selector

Seleciona o elemento com o id específico. Exemplo:

#box-id {}

. class selector

Seleciona os elementos com a classe específica. Exemplo:

.box-class {}

Seletores Especiais

'   'Descendant selector

O seletor elemento elemento é usado para selecionar todos os elementos descendentes.Exemplo:

div div {}

Descendant selector

> Direct selector

Seleciona os elementos filhos diretos.Exemplo:

div > div {}

Direct selector

+ Adjacent siblin selector

Seleciona os elementos irmãos adjacentes ao elemento. Exemplo:

.box-class + .box {}

~ General siblin selector

Seleciona todos os irmãos do elemento. Exemplo:

.box-class ~ .box {}
